# Malawi ESBL-producing *E. coli* genomics

## Data and code repository for study on clonal structure and inter-household networks of gut colonising ESBL-producing *E. coli* in rural Malawi

The data required to run the [analysis](https://amoreo71.github.io/malawi_esbl_e_coli_genomics/code/Malawi_ESBL.html) in R can be found within the 'data' folder.

This work is available as a preprint on *bioRxiv*:

*Angus M. O’Ferrall, David Lally, Peter Makaula, Gladys Namacha, Joseph M. Lewis, Patrick Musicha, Richard N. Goodman, Ellie Allman, Sabrina Moyo, Claire S. Waddington, Sekeleghe A. Kayuni, Nicholas A. Feasey, Janelisa Musaya, J. Russell Stothard, Adam P. Roberts*

[**Whole-genome sequencing reveals inter-household networks of gut-colonising ESBL- producing Escherichia coli in two rural Malawian districts**]([https://www.microbiologyresearch.org/content/journal/mgen/10.1099/mgen.0.001480](https://www.biorxiv.org/content/10.64898/2026.02.11.705350v1.full))
